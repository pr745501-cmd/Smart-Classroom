About Smart Classroom

Smart Classroom is a full‑stack web application developed using Angular, Node.js, Express & MongoDB. The system provides role‑based dashboards to ensure that each user gets access only to the features relevant to them.

This project demonstrates real‑world concepts like authentication, authorization, protected APIs, and scalable frontend architecture.
